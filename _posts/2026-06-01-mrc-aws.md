Packet trimming is a technique that allows switches to trim packets when they are congested and only send their headers. This allows receivers to detect congestion faster and request retransmissions faster than with regular packet losses. Costin Raiciu explains that [Broadcom’s switches do offer native trimming support, and Thor Ultra implements MRC in hardware](https://www.linkedin.com/posts/costin-raiciu-9799a9_mrc-is-the-result-of-an-industry-wide-collaboration-share-7457760728427462656-3n34?utm_source=share&utm_medium=member_desktop&rcm=ACoAAAA_wFcBtrzDYok3TgEsdsGk9Lnn1hU_xlI). A more detailed description appeared in a [technical report](https://arxiv.org/abs/2605.04333).

Safari and Firefox [adjust the way they interpret the HTML, CSS and JavaScript](https://denodell.com/blog/browsers-treat-big-sites-differently) of some popular web sites like TikTok or Netflix.

The GEANT network [migrated](https://connect.geant.org/2026/05/20/inside-geants-ip-network-migration-design-automation-and-operational-transformation).

The [RFC-Editor website](https://www.rfc-editor.org/) which publishes the Request For Comments has been updated. It’s now easier to access the RFC documents.

Metanorma has released [a set of software tools](https://www.metanorma.org/) that make it easier to write contributions to various standardisation bodies.

While fat-tree topologies are very popular in datacenters, Amazon announced that it has started to use [random networks](https://www.amazon.science/blog/how-flat-is-replacing-fat-in-aws-data-center-networks?utm_term=2026-may&utm_campaign=how-flat-is-replacing-fat-in-aws-data-center-networks&utm_medium=organic-asw&utm_content=2026-5-28-how-flat-is-replacing-fat-in-aws-data-center-networks&utm_source=linkedin) in its recent datacenters.

[Cryptool](https://www.cryptool.org/en/) is an online platform that provides many resources of interest to self-educate and to teach about cryptology.
